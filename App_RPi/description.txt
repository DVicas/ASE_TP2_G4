Esta aplicação consiste no desenvolvimento de um mobile hotspot. Este hotspot guarda os mac addresses dos utilizadores na eprom. Assim que um MAC se tenha ligado ao AP o seu endereço será guardado para posteriormente fazer a verificação se quem se conectou já se tinha conectado alguma vez. Caso já se tivesse conectado então será acendido um LED verde, caso contrário, ou seja, caso seja a primeira vez que um utilizador se ligue à rede será então ligado um LED vermelho
